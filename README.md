# Étude_sante_publique

Étude sur l'alimentation et la sous-nutrition dans le monde.
Montrer s'il y a assez de nourriture produite pour nourrir la population mondiale.

Pour se projet, j'ai découpé la problématique complexe en plusieurs question plus simple
J'ai commencé par l'état actuel sur la sous-nutrition et la disponibilité alimentaire
Puis j'ai montré combien de personnes pouvaient être nourrit théoriquement, basé seulement sur les calories consommé par jour
Et enfin j'ai poursuivi avec l'exemple d'un autre apport nutritionnels : les protéines

Je suis arrivé à la conclusion que si on se base uniquement sur les calories on produit plus de nourriture que nécessaire mais si on prend un autre apport nutritionnel, on peut ne pas produire assez pour tout le monde. Il faut donc prendre en compte touts les apports.
